# Hospital Infections Analysis App

A Streamlit app for analyzing Hospital Infections data.  
Upload your CSV file to view the first rows, general statistics,  
and interactive visualizations including histograms, boxplots, and correlation heatmaps.

## Features

- Upload CSV file
- View first 5 rows of the data
- View general statistics for numeric columns
- Interactive visualizations:
  - Histogram
  - Boxplot
  - Correlation heatmap
- Three tabs interface for easy navigation:
  1. Data
  2. General Statistics
  3. Visualizations

## How to Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/hospital_infections_app.git
cd hospital_infections_app
streamlit run my_app.py
